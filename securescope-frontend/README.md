# SecureScope Frontend

This is the Angular frontend for SecureScope.

## Run Locally

```powershell
npm install
npm start
```

The app runs at `http://localhost:4200` and calls the backend at `http://localhost:8080`.

## Build

```powershell
npm run build
```

## Test

```powershell
npm test -- --watch=false --browsers=ChromeHeadless
```
